Detailed Description

This file provides APIs for PluginPlay (and consumers of PluginPlay) to determine what compilation options have been enabled. These functions are such that you can make runtime decisions based on them. These functions are NOT suitable for compiletime decision making, i.e. template meta-programming. Internally PluginPlay defines src/pluginplay/config/config_impl.hpp which can be used for TMP.

N.B. Having this header file be auto-generated by the build-system would allow it to be used in TMP.
