Detailed Description

template<typename T, typename Op>
struct pluginplay::bounds_checking::CompareTo< T, Op >

Implements a functor capable of being used as a bounds check with the pluginplay.

This class wraps a binary functor and the value to compare to. In practice this binds the wrapped value to the second argument of the binary op. While binding the argument to the second argument may seem odd at first, consider an example where we create an instance fxn:

CompareTo<double, std::greater<double>> fxn(3.14);
//Is 2.0 > 3.14
assert(!fxn(2.0));
//Is 4.1 > 3.14;
assert(fxn(4.1)):

As can be seen it's the nature of this class that we are binding the right argument to the functor.

TThe type of the values being compared
