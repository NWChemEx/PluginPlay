Detailed Description

Wraps std::type_index in a class suitable for Python bindings.

To avoid potential compatibility issues with other Python bindings which may eventually export std::type_index/std::type_info we have written this small wrapper class. This class is really only intended for use with the C++/Python interface of PluginPlay as Python users should have no need to access the C++ type and C++ users should be using std::type_index.
