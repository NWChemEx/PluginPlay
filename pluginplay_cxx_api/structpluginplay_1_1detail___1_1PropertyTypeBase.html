template<typename T>
struct pluginplay::detail_::PropertyTypeBase< T >

Struct for determining what property type a property type derives from.

This is the primary template for the PropertyTypeBase class. The PropertyTypeBase class is charged with determining what property type a particular property type inherits from. The primary template is not defined, which in turn will result in a compile-time error if a user tries to instantiate it with a type that the primary template is not specialized for.
