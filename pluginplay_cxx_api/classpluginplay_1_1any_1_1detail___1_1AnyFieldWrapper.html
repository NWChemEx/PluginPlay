Detailed Description

template<typename T>
class pluginplay::any::detail_::AnyFieldWrapper< T >

Given knowledge of type T, implements AnyFieldBase.

AnyFieldBase defines the type-erased API for working with the wrapped value. The implementation of most functions occurrs in this class since it has access to the object's type and can perform the operations in a type-safe manner.

Template Parameters
TThe type of the wrapped object. Let U = std::decay_t<T>, then T can be: U, const U, or const U&. U must be copyable, comaprable via operator== and operator<, and serializable.

template<typename U , typename = disable_if_any_field_wrapper_t<std::decay_t<U>>>
pluginplay::any::detail_::AnyFieldWrapper< T >::AnyFieldWrapper ( U &&  value2wrap)

Makes an AnyFieldWrapper by wrapping the provided value.

Template Parameters
UType of the value we are wrapping. U must be implicitly convertible to T.
<anonymous>Used to disable this function via SFINAE when U is an instantiation of an input wrapper.

This ctor wraps the provided value in a newly created AnyFieldWrapper. Exactly how this is done depends on how value2wrap is passed and the exact type of T.

  • When T is a value or const value the resulting AnyFieldWrapper instance will own the wrapped value. This means the wrapped value will be a copy of value2wrap if value2wrap was passed by reference and it will take ownership of value2wrap if it was passed by value or rvalue.
  • When T is a const reference AnyFieldWrapper will only alias value2wrap and the user is responsible for keeping value2wrap alive for the lifetime of the AnyFieldWrapper.

N.B. if U is not an implicityly convertible to type T you will get a compiler error saying there is no valid any ctor for U.

Parameters
[in]value2wrapThe object being passed as an input.
Exceptions
???If wrapping value2wrap in an any throws. Same throw guarantee.

Implements AnyFieldBase::print.

This function implements AnyFieldBase::print by determining if std::ostream::operator<< is defined for type T. If it has it simply calls the stream insertion operator on the wrapped value. If it has not been defined then this function will print the name of the type and the address of the wrapped object.
