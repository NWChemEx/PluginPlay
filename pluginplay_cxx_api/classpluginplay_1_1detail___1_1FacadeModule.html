Detailed Description

template<typename PropertyType>
class pluginplay::detail_::FacadeModule< PropertyType >

Implements a facade module.

A facade module simply returns a value. Notably this means it has no body and it ignores the inputs (options and submodules) provided to it. This is useful in at least two scenarios:

  • unit testing: we don't need to hook-up a full fledged module, but can simply make a facade module to mimic the call.
  • user input: sometimes the user just wants to specify what a module returns

PropertyTypethe property type that the module satisfies.

Implements the run_as function for the Module class.

This function is what is called by the Module class's run_as and run functions when the Facade module is actually run. This function simply returns the arguments that the user provied at construction as results.
