Detailed Description

template<typename T>
struct pluginplay::bounds_checking::InRange< T >

Functor for determining if a value is in a specified range.

This class takes two values, call them low and high and is then usable as a functor for determining if a value is included in the range [low, high). That is the low-end is inclusive and the high-end is exclusive, following normal C++ conventions.

Template Parameters
TThe type of the values to compare.
