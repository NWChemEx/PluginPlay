Detailed Description

Stub class so PluginPlay can compile w/o Pybind11 support.

This class simply defines the pieces of the PythonWrapper class needed by the AnyField so that we don't have to #ifdef off code in the AnyField. Trying to call any of the methods in this version of PythonWrapper will result in runtime errors.
