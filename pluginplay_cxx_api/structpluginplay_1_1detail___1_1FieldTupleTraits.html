Detailed Description

template<typename ElementType, typename... FieldTypes>
struct pluginplay::detail_::FieldTupleTraits< ElementType, FieldTypes >

Responsible for working out the types to use in a FieldTuple instance.

This class serves largely to decouple the FieldTuple class from the process of determining its types.

Template Parameters
ElementTypeThe type used to hold the actual input or result values. It is assumed to be either ModuleInput or ModuleResult.
FieldTypesThe types of the fields in the order they are accepted or returned. Can be 0 or more types.
