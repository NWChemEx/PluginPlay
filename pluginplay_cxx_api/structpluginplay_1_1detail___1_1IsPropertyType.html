Detailed Description

template<typename T, typename BasePT = pluginplay::detail_::BasePropertyType>
struct pluginplay::detail_::IsPropertyType< T, BasePT >

Determines if type T is the type of a property type.

All property types are implemented via CRTP. Thus if T is the type of a property type it must inherit from PropertyType<T, U> where U is either:

  • BasePropertyType if T is a base property type, or
  • another type which is itself a property type.

The type IsPropertyType<T> will contain a static member value, the value of which indicates whether or not T is a a property type.

Parameters
TThe type whose property-type-ness is being questioned. T is expected to be a clean type (i.e. no pointer, reference, or const)
BasePTThe property type that T might inherit from. Default is BasePropertyType
