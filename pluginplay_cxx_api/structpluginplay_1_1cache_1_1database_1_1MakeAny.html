Detailed Description

template<typename T>
struct pluginplay::cache::database::MakeAny< T >

Specialize to change how an object of type T gets type-erased.

The TypeEraser database uses the MakeAny class to type-erase objects of type T. Generally speaking the type-erasuer is done by calling any::make_any_field, but users can specialize this class for types when there is a better way to do the conversion.

N.B. Specializations are expected to have a function convert which takes an object of type U (U is const T&, or just T) and returns an object of type AnyField.
